Shapes

A number of calculations in the game are easiest abstracted using simple shapes. Rather than everyone making their own, there are some that are built in to the Engine and have certain advantages.

Position

A Position is simply an [x, y] pair.

the x coordinate.

the y coordinate.

y : number int y

The y coordinate.

Adds together two Positions.

static sub(l : Position, r : Position) : Position Position operator - (Position r)

Subtracts one Position from another.

static equal(l : Position, r : Position) : bool bool operator == (Position r)

Checks if two Positions are the same

Checks if a Position lies inside of a Rectangle or Circle
