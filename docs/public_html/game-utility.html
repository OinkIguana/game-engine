GameUtility

The GameUtility is where most control over the state of the game comes from.

Interface

The current size of the canvaswindow.

  1. Rectangle view()
  2. void view(Position center, bool constrain)
  3. void view(Rectangle bounds, bool constrain)

Get or set the current viewport into the room.

center : Position Position center
the position that should be in the center of the view. Current viewport dimensions are preserved.
bounds : Rectangle Rectangle bounds
the area of the room that should be visible in the view.
constrain : bool bool constraindefault: true
Whether the view should be constrained within the room boundaries (default), or be allowed to look outside the room. If a dimension is larger than the room with constrain turned on, the room is centered in that dimension.
room.goto(Rm : class) template<typename Rm> void room_goto()

Closes the current Room, and moves on to the next one. If multiple layers are open, only the top (active) layer is changed.

Rm : class typename Rm
the Room to start next.
room.overlay(Rm : class) template<typename Rm> void room_overlay()

Opens the given room in a new layer, over top of the current one. The current room becomes disabled.

Rm : class typename Rm
the Room to open.
room.close() void room_close()

Closes the currently active (top) layer. If the last layer is closed an error is thrown.

end() : void void end()

Ends the game, closing the game window. Not very useful in the JavaScript version, as the game window can't really be closed.

restart() : void void restart()

Ends the game and then starts it again, using the same instance of the Engine.

sound(name : string) : Sound Sound & sound(std::string name)

Gets a reference to the Sound with the given name. If the requested Sound has not been loaded, an error is thrown. Persistent {{GameObject}}s who carry a Sound reference between Rooms that do not each individually load the sound are not permitted, and their behaviour is undefined.

name : string std::string name
the name of the Sound to find.
music(name : string) : void void music(std::string name)

Changes the current background music to the one given. The music will loop until it is changed again. If a Room is entered that does not have the currently playing music loaded, playback will be stopped.

name : string std::string name
the name of the music resource to play.
