GameEvent

The {{GameEvent}} structure is used to represent the different events that occur in the game. Though mainly used internally, there are occasions in more advanced use when a {{GameEvent}} may make it to the surface.

Interface

type : string int type
The type of event that was recorded. The value of the string is an event which exactly matches one of the event handler method names. The value of the integer is one of the values in the event type enum.
data : any[] std::vector<int> data
Some data about what exactly happened to trigger this event.

Event Types

'quit' Event::Quit
A Quit event. Ends the game, no matter how you handle it.
'gamestart' Event::GameStart
A Game Start event. No data.
'gameend' Event::GameEnd
A Game End event. No data.
'roomload' Event::RoomLoad
A Room Load event. Data holds two room classesids: the room just closed, and the room currently loading.
'roomstart' Event::RoomStart
A Room Start event. Data holds two room classesids: the room just closed, and the room that is starting.
'roomend' Event::RoomEnd
A Room End event. Data holds two room classesids: the room just closed, and the room that is starting.
'stepstart' Event::StepStart
A Step Start event. No data.
'step' Event::Step
A Step event. No data.
'stepend' Event::StepEnd
A Step End event. No data.
'keydown' Event::KeyDown
A Key Down event. Data holds an identifier for which key was pressed down.
'keyup' Event::KeyUp
A Key Up event. Data holds an identifier for which key was released.
'mousedown' Event::MouseDown
A Mouse Down event. Data holds an identifier for which mouse button was pressed down.
'mouseup' Event::MouseUp
A Mouse Up event. Data holds an identifier for which mouse button was released.
'mousemove' Event::MouseMove
A Mouse Move event. Data holds two numbers, which correspond to the new x and y position of the mouse cursor.

Example

        
          import { Room, @override } from 'engine';

          class MouseOnly extends Room {
            @override
            proc(event) {
              switch(event.type) {
                case 'keydown':
                case 'keyup':
                  // suppress keydown events in this room
                  break;
                default:
                  // do normal things for other events
                  super.proc(event);
                  break;
              }
            }
          }

          export default MouseOnly;
        
      
        
          #include <engine.h>

          class MouseOnly : public Room {
            virtual void proc(const Event & event) override {
              switch(event.type) {
                case Event::KeyDown:
                case Event::KeyUp:
                  // suppress keydown events in this room
                  break;
                default:
                  Room::proc(event);
                  break;
              }
            }
          }
