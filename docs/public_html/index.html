Game Engine

This game engine provides a number of interfaces and base classes, which work together with an instance of the Engine to provide an easy way of managing your game. Currently, this engine is available for HTML5 in JavaScript and in C++ for cross-platform desktop use.

Notice that, although these documents refer to the Engine classes only by name, in the C++ version they are all contained in the Game namespace.
